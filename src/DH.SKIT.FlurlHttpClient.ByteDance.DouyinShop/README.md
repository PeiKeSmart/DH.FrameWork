基于 `Flurl.Http` 的抖店开放平台 API 客户端。

---

### 【功能特性】

-   基于抖店开放平台 API 封装。
-   提供了抖店开放平台 API 所需的 HMAC-SHA-256、MD5 等算法工具类。
-   提供了解析回调通知事件等扩展方法。
